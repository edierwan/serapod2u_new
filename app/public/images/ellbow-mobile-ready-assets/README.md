# Ellbow Mobile Ready Assets

These files are prepared for the Ellbow RoadTour mobile redesign.

## Recommended use
- Use `webp/` in production for smaller downloads.
- Keep `png/` as transparent source files and for high-resolution screens.
- Use `reference/ellbow-mobile-ui-reference.png` only as the design reference, not as a UI background.

## Suggested repository folder
`/public/brands/ellbow/mobile/`

## Key asset mapping
- Hero mascot: `01-ellbow-cat-mascot-full`
- Verified badge: `04-ellbow-verified-shield-cat`
- Home: `05-ellbow-home-icon`
- Rewards: `06-ellbow-rewards-gift-icon`
- Scan: `07-ellbow-scan-icon`
- Product: `08-ellbow-product-pack-icon`
- Profile: `09-ellbow-profile-avatar-icon`
- Points: `11-ellbow-points-reward-icon`
- Hero background: `15-ellbow-mobile-hero-background`

The PNG files use real transparency. The checkerboard visible in some image viewers represents transparent pixels and is not part of the image.
