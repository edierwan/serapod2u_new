ELLBOW COLLECT POINTS POPUP ASSET PACK
======================================

Approved Ellbow mascot style
- white cat with teal markings
- closed smiling eyes
- pink paw pads
- red collar with gold heart tag

Recommended repo folder
app/public/images/ellbow-mobile-ready-assets/collect-points-popup/

Use webp-mobile files in the application. PNG files are transparent editing/fallback versions.

Layer order
1. CSS teal/mint hero background
2. 04 motion trail
3. 01 jumping cat
4. 02 paw coins (repeat at different sizes)
5. 03 confetti and 06 sparkles
6. white curved form panel built in CSS
7. form inputs/buttons built in HTML/CSS
8. real close button; 05 may be used only as visual reference

Important
- Do not use the full reference image as the interactive page.
- Decorative layers: pointer-events:none.
- Animate cat lightly; confetti runs once only.
- Support prefers-reduced-motion.
- Do not replace mascot with a realistic or large-eyed cat.
