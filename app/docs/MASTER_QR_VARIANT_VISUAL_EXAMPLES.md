# Master QR Codes - Variant Column Visual Examples

## Quick Reference

**New Column:** "Variant" (Column E, width 50)

**Format:**
- Single variant: `Product Name - Variant Name`
- Mixed variants: `MIXED: Variant1 + Variant2 + ...`
